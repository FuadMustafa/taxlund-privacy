1
Tətbiqin Məqsədi
Purpose of the Extension
Purpose of the Extension
Tətbiqin Məqsədi

Tax Lund, Azərbaycan Dövlət Vergi Portalı (new.e-taxes.gov.az) istifadəçilərinin öz hesablarındakı EQF (Elektron Qaimə Fakturu) məlumatlarını rahat şəkildə toplamalarına, cədvəl formatında görüntüləmələrinə və Excel (.xlsx) formatında ixrac etmələrinə kömək edir. Tətbiq yalnız istifadəçinin öz hesab sessiyası daxilindəki məlumatlarla işləyir.

Tax Lund helps users of the Azerbaijan State Tax Portal (new.e-taxes.gov.az) to conveniently collect their own EQF (Electronic Invoice) data, view it in a table format, and export it to Excel (.xlsx). The extension works exclusively with data within the user's own account session.

How We Use Data

The data we collect is used solely for the following purposes:

4
Məlumat Saxlama Yeri
Data Storage Location
Data Storage Location
Məlumat Saxlama Yeri
💾
chrome.storage.local — Yalnız Yerli Saxlama chrome.storage.local — Local Storage Only

Bütün EQF fatura məlumatları, istifadəçi profil konfiqurasiyaları və panel vəziyyəti yalnız sizin cihazınızda Chrome-un yerli yaddaşında saxlanılır. Bu məlumatlar heç bir bulud serverinə, bizim tərəfimizdən idarə edilən serverə və ya başqa xarici yerə göndərilmir.

All EQF invoice data, user profile configurations, and panel state are stored only on your device in Chrome's local storage. This data is not sent to any cloud server, server managed by us, or any other external location.

Məlumatların silinməsi: Tax Lund Chrome əlavəsini brauzerdən sildikdə, bütün yerli məlumatlar (chrome.storage.local) Chrome tərəfindən avtomatik olaraq tamamilə silinir. İstifadəçi istənilən vaxt əlavə daxilindəki "🧹 Yaddaşı Təmizlə" düyməsindən yerli məlumatları silə bilər.

Data deletion: When the Tax Lund Chrome extension is removed from the browser, all local data (chrome.storage.local) is automatically and completely deleted by Chrome. Users can delete local data at any time using the "🧹 Clear Memory" button inside the extension.

5
Xarici Servis: Autentifikasiya Serveri
External Service: Authentication Server
External Service: Authentication Server
Xarici Servis: Autentifikasiya Serveri

Tax Lund giriş zamanı istifadəçi adı və şifrənin SHA-256 hash-ını öz autentifikasiya serverimizə (Google Apps Script üzərindən idarə edilən) HTTPS vasitəsilə göndərir.

During login, Tax Lund sends the username and the SHA-256 hash of the password to our authentication server (managed via Google Apps Script) over HTTPS.

🔐
Şifrə qoruması Password protection

Şifrəniz heç vaxt açıq mətn (plain text) kimi göndərilmir. Göndərilmədən əvvəl SHA-256 kriptoqrafik hash funksiyası ilə cihazınızda işlənir. Auth serverimiz yalnız bu hash dəyərini alır; orijinal şifrəni heç vaxt görüb saxlamır.

Your password is never sent in plain text. It is processed on your device with the SHA-256 cryptographic hash function before being sent. Our auth server only receives this hash value; it never sees or stores the original password.

Autentifikasiya serveri yalnız giriş icazəsini yoxlayır; EQF məlumatları, fatura nömrələri və ya VÖEN nömrələri auth serverinə heç vaxt göndərilmir.

The authentication server only verifies login permission; EQF data, invoice numbers, or TIN numbers are never sent to the auth server.

6
Portalın JWT Token-ının Emalı
Portal JWT Token Handling
Portal JWT Token Handling
Portalın JWT Token-ının Emalı

Tax Lund, istifadəçinin new.e-taxes.gov.az portalına öz brauzerindən giriş edərək əldə etdiyi sessiya token-ını (JWT) portal səhifəsinin localStorage-ından oxuyur.

Tax Lund reads the session token (JWT) that the user obtained by logging into the new.e-taxes.gov.az portal from their own browser, from the portal page's localStorage.

🔒
Bu token necə istifadə edilir? How is this token used?

Token yalnız eyni portalın (new.e-taxes.gov.az) API uç nöqtələrinə istifadəçi adından sorğu göndərmək üçün istifadə edilir. Token heç bir xarici serverə, üçüncü tərəfə və ya bizim serverlərimizə göndərilmir. İstifadəçi artıq bu portal üçün icazəli olduğu məlumatlarına avtomatik giriş əldə edir — əlavə heç bir bağımsız giriş hüququ almır.

The token is used exclusively to send requests on behalf of the user to the API endpoints of the same portal (new.e-taxes.gov.az). The token is never sent to any external server, third party, or our servers. The user gains automated access to data they are already authorized for on this portal — the extension does not acquire any additional independent access rights.

Third-Party Sharing

EQF data collected by the user, TIN numbers, invoice amounts, and any other personal financial data is never shared, sold, or transferred with advertising companies, analytics services, data brokers, or any other third parties.

Children's Privacy
Children's Privacy
Uşaqların Məxfiliyi

Tax Lund yalnız e-taxes.gov.az portalında hesabı olan vergi ödəyiciləri tərəfindən istifadə üçün nəzərdə tutulub. Bu portal üzrə yaş məhdudiyyəti Azərbaycan Respublikasının qanunvericiliyinə uyğundur. Tətbiq bilərəkdən 18 yaşından kiçik şəxslərdən məlumat toplamır.

Tax Lund is intended for use only by taxpayers with an account on the e-taxes.gov.az portal. The age restriction for this portal is in accordance with the legislation of the Republic of Azerbaijan. The extension does not knowingly collect data from persons under 18 years of age.
